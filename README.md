# ATM-simulation

## Описание

ATM Simulator — консольное приложение на Python, моделирующее базовые операции банкомата.  
Проект демонстрирует работу с пользовательским вводом, условными конструкциями и управлением состоянием (баланс, PIN-код).  
Используется как учебный пример простой финансовой системы.

## Цель проекта

- Отработка базовых конструкций Python  
- Моделирование логики банковских операций  
- Работа с интерфейсом командной строки  
- Формирование читаемого и структурированного кода  

## Функциональность

- Проверка баланса  
- Снятие средств с проверкой доступного остатка  
- Пополнение счета  
- Изменение PIN-кода (при наличии)  
- Обработка некорректного ввода  

## Особенности реализации

- Программа работает в цикле взаимодействия с пользователем  
- Используются условные операторы (`if/elif/else`)  
- Данные (баланс) хранятся во время выполнения программы  
- Реализован текстовый интерфейс
