ATM Simulation

Описание

ATM Simulator — консольное приложение на Python, моделирующее базовые функции банкомата.
Проект демонстрирует работу с пользовательским вводом, условными конструкциями, циклами и управлением состоянием данных (баланс, PIN-код).

Проект создан в учебных целях для закрепления основ программирования на Python.


---

Цель проекта

Изучение базового синтаксиса Python

Практика работы с пользовательским вводом

Использование условных операторов и циклов

Формирование навыков структурирования кода

Понимание работы с изменяемым состоянием программы



---

Функциональность

Проверка баланса

Пополнение счета

Снятие средств с проверкой доступного остатка

Проверка PIN-кода (при наличии реализации)

Обработка некорректного пользовательского ввода



---

Особенности реализации

Программа работает в цикле до выхода пользователя

Логика разделена на функции

Используются условные конструкции (if / elif / else)

Баланс хранится в памяти во время выполнения программы

Реализован текстовый интерфейс взаимодействия с пользователем

Установка и запуск

Клонирование репозитория

Переход в директорию проекта

Запуск программы
